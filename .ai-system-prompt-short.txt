# ihr360-xxx-static Project Instructions

React 18 + TypeScript + Rsbuild + single-spa microfront-end app.

## UI Framework: Tailwind CSS v4 + shadcn-ui

**CRITICAL: NO antd4, NO ihr360-web-ui3, NO CSS files**

## Rules

1. **Styling**: ONLY use Tailwind utility classes
   ```tsx
   // ✅ CORRECT
   <button className="px-4 py-2 bg-primary text-primary-foreground rounded-md">

   // ❌ FORBIDDEN
   import { Button } from 'antd4';
   import './styles.css';
   ```

2. **Imports**: Use @ alias
   ```tsx
   import { Button } from '@/components/ui/button';
   import { cn } from '@/lib/utils';
   ```

3. **Theme Colors**: Primary #26c2a4
   - Use: `bg-primary`, `text-primary`, `bg-secondary`, `bg-destructive`, `border-border`

4. **shadcn-ui**: For complex components
   ```bash
   pnpm dlx shadcn@latest add button
   ```

5. **Responsive**: Use breakpoints
   ```tsx
   <div className="w-full md:w-1/2 lg:w-1/3">
   ```

## Component Pattern
```tsx
export function Card({ title }: { title: string }) {
  return (
    <div className="p-6 bg-white rounded-lg shadow">
      <h3 className="text-lg font-semibold">{title}</h3>
    </div>
  );
}
```

## Forbidden
- ❌ antd4, ihr360-web-ui3
- ❌ CSS/Less files
- ❌ @apply directive
- ❌ Inline styles (rare exceptions only)

## Priority
1. Tailwind classes (first)
2. shadcn-ui components
3. CVA for variants
4. Direct CSS (last resort)
